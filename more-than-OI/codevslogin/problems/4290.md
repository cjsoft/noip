# 4290 二元一次不定方程   
### 时间限制： 1 s     空间限制： 1000 KB     题目等级： 青铜  
## 题目描述：  

<pre>
背景：  
    <null>_(^q^/∠)_   
  
描述：  
    已知a,b,c, 求满足ax+by=c的整数对(x,y)中x为正整数且最小；    若此时0<=x,y，则输出x y，若此时y<0则输出“sometimes naive”（没有双引号）；    若没有整数对(x,y)满足条件则输出“too simple”
</pre>
  
  
## 输入描述：  

<pre>
输入  
    一行 三个数，a，b，c；   

</pre>
  
  
## 输出描述：  

<pre>
输出    一行，x y或too simple或sometimes naive 
</pre>
  
  
## 样例输入：  

<pre><code>
    #1  
    0 0 0  
    #2  
    2 0 1  

</code></pre>
  
  
## 输出输出：  

<pre><code>
    #1    0 0        #2    too simple  

</code></pre>
