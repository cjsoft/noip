# 1871 下落的圆盘   
### 时间限制： 1 s     空间限制： 256000 KB     题目等级： 大师  
## 题目描述：  

<pre>
有n个圆盘从天而降，后面落下的可以盖住前面的。求最后形成的封闭区域的周长。 看下面这副图, 所有的红色线条的总长度即为所求.
</pre>
  
  
## 输入描述：  

<pre>
n ri xi y1 ... rn xn yn
</pre>
  
  
## 输出描述：  

<pre>
最后的周长，保留三位小数
</pre>
  
  
## 样例输入：  

<pre><code>
2  
 1 0 0  
 1 1 0
</code></pre>
  
  
## 输出输出：  

<pre><code>
10.472
</code></pre>
  
  
## 数据范围及提示：  

<pre>
n<=1000
</pre>
