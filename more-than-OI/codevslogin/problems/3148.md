# 3148 矩阵A+B   
### 时间限制： 1 s&nbsp;&nbsp;&nbsp;&nbsp;空间限制： 4000 KB&nbsp;&nbsp;&nbsp;&nbsp;题目等级： 黄金  
## 题目描述：  

<pre>
进入XX中学的第二天，老师又给我们出了A+B PROBLEM
但这次的A+B PROBLEM 难度降(ti)低(gao)了
这回读入两个矩阵A,B输出结果.
</pre>
  
  
## 输入描述：  

<pre>
第一行n 表示a b 的行列数
接下来n行表示矩阵a
接下来n行表示矩阵b
</pre>
  
  
## 输出描述：  

<pre>
n行为两个矩阵的和
</pre>
  
  
## 样例输入：  

<pre>
2
1 1
1 1
2 2
2 2
</pre>
  
  
## 样例输出：  

<pre>
3 3
3 3
</pre>
  
  
## 数据范围及提示：  

<pre>
行数列数相同，且均<=100.
矩阵里的数字不超过int型
空间开的很紧（4M），小心。
</pre>
