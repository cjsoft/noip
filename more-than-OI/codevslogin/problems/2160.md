# 2160 进制转换   
### 时间限制： 1 s&nbsp;&nbsp;&nbsp;&nbsp;空间限制： 32000 KB&nbsp;&nbsp;&nbsp;&nbsp;题目等级： 黄金  
## 题目描述：  

<pre>
对于十进制整数N，试求其-2进制表示。  
例如，因为 1*1 + 1*-2 + 1*4 + 0*-8 +1*16 + 1*-32 = -13 ，所以(-13)_10 = (110111)_-2。
</pre>
  
  
## 输入描述：  

<pre>
一个整数，代表要转换的十进制数。
</pre>
  
  
## 输出描述：  

<pre>
一个整数，代表N的-2进制表示。
</pre>
  
  
## 样例输入：  

<pre>
-13
</pre>
  
  
## 样例输出：  

<pre>
110111
</pre>
  
  
## 数据范围及提示：  

<pre>
|N| <= 2000000000
</pre>
