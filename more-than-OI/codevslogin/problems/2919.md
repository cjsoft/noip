# 2919 选择题   
### 时间限制： 1 s&nbsp;&nbsp;&nbsp;&nbsp;空间限制： 16000 KB&nbsp;&nbsp;&nbsp;&nbsp;题目等级： 黄金  
## 题目描述：  

<pre>
某同学考试，在N*M的答题卡上写了A,B,C,D四种答案。
他做完了，又不能交，一看表，离打铃还有N久。
他开始玩一个游戏：选一个格子X,Y，从这个格子出发向4个方向找相同的选项，找到的再如此。
求形成的图形的面积。（一个选项占一个单位面积）
</pre>
  
  
## 输入描述：  

<pre>
N M X  Y
答题卡（矩阵）
</pre>
  
  
## 输出描述：  

<pre>
面积
</pre>
  
  
## 样例输入：  

<pre>
3 3 1 2
A C B
C C C
D C A
</pre>
  
  
## 样例输出：  

<pre>
5
</pre>
  
  
## 数据范围及提示：  

<pre>
N,M<=15.
对于33%数据，只有A。
</pre>
