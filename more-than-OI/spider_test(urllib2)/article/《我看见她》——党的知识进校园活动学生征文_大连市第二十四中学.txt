<div style="text-align: center">
	一<br/>
	 <br/>
	我闭上眼<br/>
	看见她<br/>
	她是一面燃烧的墙<br/>
	她是几首飘荡的歌<br/>
	她是万束惊异的光<br/>
	她是惊雷<br/>
	她也是清波<br/>
	 <br/>
	二<br/>
	 <br/>
	延安城里<br/>
	我看见了她<br/>
	镰刀与锤子拥抱出麦黄的希望<br/>
	井冈山头<br/>
	我看见了她<br/>
	号角和战歌沾满了春天的曙光<br/>
	 <br/>
	金沙江边<br/>
	有她纤纤玉足踏过的痕迹<br/>
	大渡河上<br/>
	仍留她声声呼喊的印记<br/>
	 <br/>
	她是清眸淡墨的<br/>
	在南湖的画舫间冥想<br/>
	她是坚韧赤诚的<br/>
	在南昌的战火里歌唱<br/>
	 <br/>
	她以孩童的双眼<br/>
	凝望脚下的土地<br/>
	她稚嫩的双手<br/>
	端起沉甸甸的枪<br/>
	 <br/>
	我看见她的痛<br/>
	她浅浅的呻吟<br/>
	我看见她赤着脚<br/>
	含着血水奔向远方<br/>
	 <br/>
	我看见她的戎装<br/>
	融化她青春的容貌<br/>
	田里，乡间<br/>
	我看见她的儿女<br/>
	他们贫瘠<br/>
	但他们流淌着最诚挚的热血<br/>
	 <br/>
	三<br/>
	 <br/>
	我睁开眼<br/>
	看见她<br/>
	看见她就在这里<br/>
	她匹敌不了千军万马<br/>
	却用坚毅的灵魂<br/>
	哺育一个伤痕累累的民族<br/>
	 <br/>
	她没有轻薄的承诺<br/>
	只有默默徜徉<br/>
	徜徉在崭新的征途<br/>
	她是巨人<br/>
	她也是母亲<br/>
	 <br/>
	四<br/>
	 <br/>
	她熄灭了硝烟<br/>
	升起袅袅炊烟<br/>
	她是碗中一粒珍贵的米<br/>
	她是林立新楼的一片瓦<br/>
	 <br/>
	她如歌者<br/>
	在西部戈壁里挥舞衣袖<br/>
	她如春风<br/>
	抚过小岗村受伤的羽翼<br/>
	 <br/>
	她在废墟的坍塌处<br/>
	心疼地落下泪水<br/>
	她在决堤的洪水处<br/>
	用身体抵住浪潮<br/>
	 <br/>
	我看见她的魂魄<br/>
	在天宫<br/>
	载着神舟荡漾<br/>
	我看见她在星宇<br/>
	舒展了紧皱的眉头<br/>
	 <br/>
	我欣喜地<br/>
	发现她的身躯不再干瘪<br/>
	她的微笑不再苦涩<br/>
	 <br/>
	她苏醒了<br/>
	带着新纪元的眸光<br/>
	她成熟了<br/>
	在骄傲的龙背上竖起了腰肢<br/>
	 <br/>
	五<br/>
	 <br/>
	她是党<br/>
	是我们生命中最温暖的臂弯<br/>
	 <br/>
	她给我一个世界<br/>
	她许我一个未来<br/>
	 <br/>
	我不曾见过她<br/>
	但她就在这里<br/>
	她的故事<br/>
	沉淀出希望的味道<br/>
	 <br/>
	我看见党<br/>
	看见一个崭新的中国<br/>
	看见我，一个高中生，即将走向的世界<br/>
<strong>我们爱党</strong><br/>
<strong>以最真诚的名义</strong><br/>
<br/>
	 </div>
<div style="text-align: right">
<strong>大连市第二十四中学 郑雨知</strong><br/>
