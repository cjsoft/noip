<font face="幼圆">   <br/>
  <span style="font-size: 14px">  化学综艺活动是我校化学组组织的一项大型活动，整个活动充满着知识性、趣味性及娱乐性，是典型的寓教于乐活动，目前，它已成为我校学生最为喜欢的活动之一，同时也我校领导、广大教师及美国教育专家好评。 </span></font>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">1.活动开展的背景<br/>
<br/>
	      </font><font color="#000000" face="幼圆">21世纪，将是知识经济占国际经济主导地位的世纪。面向21世纪的教育，就是面向知识经济的教育。而创新精神和实践能力是知识经济发展的强劲动力，创新意识的强弱和实践能力的优劣成为人才素质高低的重要标志。提倡创新精神，构建创新体系，它的目的就是追求新发现、探索新规律、创造新方法和积累新知识，使中华民族永远立于不败之地。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      要想创新，首先应该有求知欲，应该有兴趣。正如江泽民总书记在第三次教育工作会议上指出的：“教育在培养民族创新精神和培养创造性人才方面，肩负着特殊的使命，每一个学校都要爱护和培养学生的好奇心，求知欲，帮助学生自主学习，独立思考，保护学生的探索精神、创新思维，营造崇尚真知，追求真理的氛围，为学生的禀赋和潜能的充分开发创造一种宽松的环境。” </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      基于培养学生的化学兴趣和好奇心，我们组织了化学棕艺活动。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">2.活动的形式 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      以班级为单位，每班一个代表队（3—5名队员），一个助威团（人数不限）。题目分必答题和抢答题。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.活动的内容 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.1 说说看（必答题） </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      围绕某个知识点去命题，将此知识点公开，并将与其有关的某些知识隐藏，参赛队员在规定时间内快速的说出这些隐藏的知识点，每说对一项便得10分（可自由规定）例如围绕“烷烃”这一知识点可设计如下的隐藏点：“碳碳单键”、“取代反应”、“饱和链烃”“CnH2n+2”等等。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.2实验包裹（必答题） </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      对基本实验操作或简单的综合实验进行编辑，设置问题，限制在规定时间能回答。 </font></span></h4>

<span style="font-size: 14px"><span lang="EN-US" style="font-family: 幼圆"><font color="#000000">3.根据什么现象证明结论                                        </font></span></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.3猜猜看（必答题） </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      本题需两个队员合作，一个背对屏幕，当屏幕显示一个项目时，面对屏幕的队员要尽可能的描述这一选项，但一定不要涉及这一选项中的语言，另一队员应快速做答，本题有规定时间。例：屏幕显示“门捷列夫”，面对屏幕的队员可提示“元素周期律”等。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.4找错误（抢答题） </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      本题可以将基本实验操作中易错误的编辑到一起，以图片的形式扫描到电脑中，一张一张快速闪过，请队员们找到错误之处。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.5化学谜语（抢答题） </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      收集一些有趣的化学谜语，既能很好的复习化学知识，又能提高能力，在笑声中学到了化学知识。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">3.6实验操作大比拼（必答题） </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      一般出一些实验鉴别题，比如：八瓶溶液分别是：氯化铁、碳酸钠、氢氧化钠、硝酸银、硫化钠、氯化钠、硫氰化钾、碘化钾—淀粉。不用其他试剂请一一鉴别。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      在比赛过程中，还可以穿插一些化学魔术、化学歌曲之类的节目。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">4.活动的思考 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      通过建立在自觉、自愿、自主基础上的活动。是化学课堂教育的延伸和补充。它寓化学教育于活动中，富于实践性和创造性，可以帮助学生积累大量的感性认识，拓宽知识面，提高实验技能，培养观察能力。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">4.1极大的激发了学生学习化学的兴趣 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      学习兴趣是学生探求知识的巨大动力，是发明创造的知识源泉。化学综艺活动是一种学生愿意参加的活动，它比课堂教学能更好的发展学生的兴趣、爱好和特长。无论是学习成绩好的同学，还是成绩差的同学，都可以通过参加化学综艺活动提高他们的自信，激发他们的求知欲引导他们在化学世界里开辟一个最喜爱的领域。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">4.2营造了广阔的思维空间 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      著名的教育家赞可夫说过：“学生积极的情感，欢乐的情绪，能使学生精神振奋，思维活跃，容易形成新的联想。而消极的情绪，则会抑制学生的智力活动。”化学综艺活动，就能营造出一个无拘无束的自由天地，让学生的思维纵横千里，任意驰骋。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">4.3开阔了学生的知识视野，引发求知欲望 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      综合化学课堂，开展丰富多采的化学综合活动，有利于学生扩大知识面，开阔视野，并是学生的幻想升华为渴求知识的热情，从而产生探求未知世界的创造动机，产生创新思维。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">4.4有利于发现学生的闪光点 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      现在的学生的知识是丰富的，精彩的，这一点在活动中体现的非常明显，作为教师应该善于发现好的苗子，当然也应该善于学习，学习学生的闪光点。 </font></span></h4>
<h4>
<span style="font-size: 14px"><font color="#000000" face="幼圆">      著名的美籍华裔科学家杨振宁教授说过：“学海无涯乐作舟”我们的化学综艺活动正向此方向迈进。 </font></span></h4>
