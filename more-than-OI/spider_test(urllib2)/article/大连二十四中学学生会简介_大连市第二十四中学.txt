<strong>                   2010届学生会概况</strong><br/>
<br/>
大连二十四中学学生会是全体学生自己的群众组织，学生会在校团委指导、帮助下，团结全体学生进行工作。学生会以全面发展、提高综合素质为目标，以学习为中心，倡导自我服务、自我管理、自我教育，组织全体学生开展丰富多彩、生动活泼的学习、科技、文艺、体育、志愿服务公益劳动等各类活动，引导同学们不断道德水平，树立立志成才、立志报国远大理想，增强劳动观念，增进身心健康，维护同学的正当权益，反映同学的建议、意见和要求，促进同学之间，同学与教职员工之间的团结协作，协助学校创造良好的教学秩序和学习生活环境，为学生的健康成长服务。<br/>
大连二十四中2010届学生会成立于2010年3月。设主席一名，下设副主席及七个部门每部部长一名，副部长一名，部员若干。现任大连二十四中学生会主席高二十班肖强由全体高一高二学生选举产生，其他学生会成员由主席及校领导组阁产生。<br/>
 <br/>
<strong>                  机构设置及相关职责:</strong><br/>
<br/>
<strong>主席团</strong>：全面负责学生会工作，了解掌握各部门情况发现问题及时采取措施，向有关部门反映，提出意见和要求。主持召开定期、不定期的学生会成员会议。研究每一阶段工作、分析情况，落实措施，总结经验教训，制订学生会工作计划。<br/>
<br/>
<strong>秘书处</strong>：搞好学生会的对外联络工作。发展同其他学校的联系与友谊。处理学生会文秘、会议记录工作，协助起草计划、总结，负责做好学生会各项活动有关资料收集。　<br/>
<br/>
<strong>学习部</strong>：明确学习部的工作职责，定期召开部属会议，研究工作，交流情况，落实措施。通过组织开展知识竞赛、辩论大赛及各种形式多样、寓教于乐的学习活动，拓宽视野，激发同学们的学习兴趣，营造良好的学习氛围，提高同学们的学习能力。<br/>
<br/>
<strong>生活部</strong>：本着“服务同学”的宗旨，协助值周教师及值周班的工作。关心同学们的学习生活状况，及时收集情况，反映意见，布置工作。为全校师生营造一个欢畅，活跃，上进的学习、生活氛围。主要负责学校日常纪律卫生等事项的管理，为维持一个良好和谐的校园环境，及大型活动组织的良好秩序做出了重要贡献。<br/>
<br/>
<strong>文艺部</strong>：丰富同学们的课余生活，配合学校团委举办各类文艺活动，组织开展校园歌手大赛、元旦联欢会等活动，丰富学生的课余文化生活。 <br/>
<br/>
<strong>宣传部</strong>: 围绕学校及学生会每个阶段的工作中心，做好学生会工作的对外宣传，负责校园海报、宣传栏的布置,同时负责各种活动的立体式宣传，让同学们了解活动的最新动态、活动构想和目的，为各项活动的正常进行提供保障,扩大学生会知名度。力争营造一个和谐、健康的校园文化。<br/>
<br/>
<strong>体育部：</strong>协助校体育组及其它部门组织广大学生开展校内群众性体育活动，如长绳比赛、篮球赛、排球赛帮助同学活跃身心，增强体质；组织各类体育比赛，倡导良好的赛风，丰富同学课余生活，<br/>
<br/>
<strong>社团部</strong>:负责协调各社团的建设和活动开展，监督并协助各社团开展活动。定期召开社团负责人会议，制订计划、总结。组织各社团成果展览。<br/>
社团部下设近二十个社团分别为：广播站、记者站、蓝帆文学社、星空杂志社、新领军外语社、爱心社、尚艺棋社、心蕾手工坊、α天文社、step up街舞工作室、SA科学社、问号影视工作室、风吟剧社、月薰漫研社、The Wing翼乐团、电视台……<br/>
<br/>
    本学年，我校学生会和大连育明高中学生会、大连八中学生会、大连一中学生会联合举办了四校联谊嘉年华活动。<br/>
我校学生会和其他各校学生会联合举办了的四校联谊嘉年华开幕式，受到同学们的一致好评。<br/>
在我校学生会主办的知识大赛上，新颖的创新模式让同学们在娱乐中拓展视野增长知识。以拓宽同学的知识面，领略古今中外名人佳作，陶冶同学的高雅情操为目的，四校选手及同学在激烈的角逐中得到了充分学习。<br/>
徒步大会让同学们呼吸了春天的气息，在绿色中感受环保与低碳。<br/>
我校参加了在一中举办的“四校联谊英语对对碰”活动，我校新领军外语社派出参赛选手，演出了精彩的英语情景剧，展示了很高的英语水平，为母校争得荣誉。我校学生会积极组织安排比赛及参赛队员，并取得了骄人的好成绩。<br/>
文艺部组织的一年一度的校园歌手大赛，由校团委主办学生会组织、承办。从海选到初赛到决赛受到同学们的高度关注。<br/>
体育部联合体育组组织的校体育节，通过足球比赛、篮球比赛和排球比赛，锻炼了同学们的体魄，丰富了同学们的生活，也增进了同学们的友谊。<br/>
学生会积极策划、设计、组织并带领同学们参与以上活动，使得学校的各项活动趣味与挑战并存，歌声与欢笑同在。每项活动在学生会的精心组织，同学们的配合下，都井然有序进行，这些不仅体现了学生会干部较强的组织管理能力，也体现了中学生的良好的精神风貌。虽然工作是忙碌的、辛苦的，但每一位学生会成员都在实践中成长，在创新中进步，在挑战中超越。一同期待我们把校园建设得更好！<br/>
