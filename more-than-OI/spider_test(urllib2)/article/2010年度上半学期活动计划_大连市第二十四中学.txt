普及天文奇观：  <br/>
春夏之交的五月，天气逐渐热了起来。随着太阳直射点逐渐北移，对于我们来说，夜晚也将变得短暂。然而，这个月的精彩天象并未减少，金星的观测条件越来越好，而继上月水星东大距之后，5月下旬它还将运行到西大距的位置，月初宝瓶座η流星雨也将上演，天文爱好者又要忙活起来了。  <br/>
<br/>
<br/>
5月6日 宝瓶座η流星雨极大  <br/>
       著名的哈雷彗星，为地球带来了两个流量较大的流星雨，分别是5月的宝瓶座η流星雨和极大出现在10月中下旬的猎户座流星雨。与前者相比，后者可观测时间更长，也更为天文爱好者熟知。  <br/>
<br/>
       宝瓶座η流星雨是因其辐射点紧邻宝瓶座η（ETA）星而得名，宝瓶座这个位于黄道上的星座，北半球秋季观测它的条件最好。由于没什么较亮的星，想必大家对它并不熟悉，平时听到最多的恐怕还是另一个错误的翻译——水瓶座。其实宝瓶座的天区面积很大，在88的星座中排行第10，黄道和天赤道穿其而过。但该星座最亮的α星亮度仅为3等左右，而η星只是一颗4等星。  <br/>
<br/>
       宝瓶座η流星雨是上半年流量较大的流星雨之一，由于辐射点几乎位于天赤道上，因此地球上绝大部分的区域都能对其进行观测。今年它的活跃期从4月19日一直持续到5月28日。在此期间，北半球，特别是中高纬度地区日出很早，辐射点地平高度并不理想时，常规的光学观测就已经无法进行了。而在南半球，该流星的观测条件要更好一些。更据国际流星组织的预报，它的极大很有可能出现在5月6日，理论上每小时辐射点天顶流量（ZHR）可达85。其实预报者对这个流量期望并不大，因此在后面加了个“*”。事实上，多数年份该流星雨的ZHR值在30至50之间，而由于辐射点地平高度的限制，我们能观测到的群内流星并不多。  <br/>
<br/>
       今年宝瓶座η流星雨极大期间，月光对观测的干扰比较严重。5月6日是阴历三月廿三，以北京地区的观测点为例，下弦月在凌晨1时左右升起，正位于宝瓶座西侧的摩羯座。流星雨的辐射点大约2时出地平，4时晨光对观测的影响就很大了，只有之前不到一个小时的时间观测条件尚可。还需要提醒大家的是，宝瓶座η流星雨的群内流星速度大约为每秒66千米，属于快速流星，。当然这也只能在一定程度上将其与其它流星相区分。 <br/>
