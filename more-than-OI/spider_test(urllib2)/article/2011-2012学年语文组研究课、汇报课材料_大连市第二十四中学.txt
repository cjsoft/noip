<p>
	    一、上课教师自评<br/>
	    （一）耿英年：《项脊轩志》说课稿<br/>
	    各位同仁：大家好！我今天说课的题目是《项脊轩志》，我从以下几方面来解说这一课。 <br/>
	    一、说教材 <br/>
	    《项脊轩志》是选修版《中国古代诗歌散文》一篇选读课文，这一单元所选文本的共同点是表达人间亲情，旨在让学生接受浓浓亲情的洗礼，提升自己的精神境界。《项脊轩志》是明代散文家归有光的代表作，属于一篇典范的浅易文言文。新课程标准对阅读浅易文言文有如下要求：“能借助注释和工具书，理解词句含义，读懂文章内容，了解梳理常见的文言实词、虚词、句式的意义、用法，注重在阅读中举一反三。”根据以上标准及这篇课文的自身特点，本课的教学重点有三个：1、学生自主梳理课文涉及的文言实词、虚词的意义、用法及各类句式。2、在熟读的基础上理清把握文章的脉络。形成整体阅读能力。3、学习本文善于捕捉生活中典型的细节和场面来表达真挚感情的写作方法。 <br/>
	    二、说目标 <br/>
	    根据新课标强调“知识与能力”、“过程与方法”、“情感态度和价值观”三位一体的课程理念以及教材分析，我把本文的教学目标确定为： <br/>
	    1、掌握阅读浅易文言文的方法，能根据要求筛选归纳文本信息。 <br/>
	    2、学会抓住生活细节来表达情感的写作方法。 <br/>
	    3、认知亲情，学会感恩。 <br/>
	    三、说教法与学情分析： <br/>
	    大部分学生把文言文阅读等同于对个别字词的理解与记忆，不懂得把文本当成一篇文章来阅读，尤其欠缺整体把握意识，本课教学中重点培养学生的整体阅读能力；学生对本文具备一定的认知基础，初中学过的《陋室铭》也是一篇以描写书斋居室入题的文章，不过一篇重在发感，一篇重在抒情而已，因此在学法上可采取自主、合作、探究的学习方式。 <br/>
	    四、说教学过程 <br/>
	    我将教学过程设计为两节课，第一节课重在文章梳理，第二堂课理清文脉，感知情感，具体有以下几个步骤： <br/>
	    1、新课导入；2、解题；3、课堂精练；4、全文的感情线索；5、一个书斋；6、两种情感；7、三世变迁；8、四个女人；9、高考链接；<br/>
	    10、总结：好端端一个大家庭,最终分崩离析,这是一悲,人们常说,人生有三大不幸:少年丧母,中年丧妻,老年丧子,归有光就占了两大不幸,再加上功名未遂,辜负了祖母的殷切期望,接二连三的悲压得他喘不过气来,只能借《项脊轩志》这一篇散文来寄托对大家庭破落的感慨和对亲人的怀念。<br/>
	    11、这篇文章对我们写作有何启示?不必写惊天动地的大事,曰常生活中的平凡小事,只要你用心感受,抓住富有特征的细节,定能人出富有真情实感的好文章。</p>
<p>
	    （二）王丹：《六国论》说课稿<br/>
	    各位老师:大家好！今天我说课的题目是《六国论》。<br/>
	    一、说教材<br/>
	    1、教学内容分析：这是一篇著名的史论，作者苏洵在文章中评论战国时期六国灭亡的原因，说明“六国破灭，弊在赂秦”，讽谏北宋王朝以六国为鉴，不要为契丹、西夏“积威之所劫”，一味贿赂以求苟安，而要奋起御侮，积极谋求“不赂而胜之”之道。文章借古讽今，切中时弊，论点鲜明，论证严密，语言生动，气势充沛。学习本文有助于学生议论文的写作，有助于学生对国家兴亡的深入思考。<br/>
	    2、教学重难点：课文分析的重点宜放在如何围绕中心论点展开论证上。解读课文可以从段落结构入手，在梳理文章脉络的基础上，把握作者所要表达的观点。<br/>
	难点在于引导学生理解分析议论文的写作。<br/>
	    3、课时安排：二课时。第一课时整体感知课文：理清文章脉络，体会作者观点。注重朗读和讨论。第二课时探讨文章内容及文章的写作技巧。<br/>
	    二、说目标<br/>
	    （一）、知识教育目标：１、学习本文论点鲜明、论证严密、结构严整的特点。２、积累文中的古今异义、词类活用等文言现象。<br/>
	    （二）、能力培养目标：培养学生梳理文章脉络的能力，学习借古讽今的写作方法。<br/>
	    （三）、德育渗透目标：体会作者关心国家、反对屈辱求和、主张抵抗外侮的感情，培养学生关心天下兴亡的历史责任感。<br/>
	    三、说学情：<br/>
	    在掌握基本文言虚实词的基础上，鼓励学生创作文言古诗词，并在课堂中予以展示。<br/>
	    四、说教法：<br/>
	    根据教学目标及重点难点,我具体选择了三种教法。1． 诵读法。通过诵读培养学生的语感，潜移默化地积累文言知识，领会文章强烈的雄辩力量和充沛的气势。2、学生相互合作讨论。质疑，激疑，引导学生思考关于国家兴亡的历史教训和文章的写作方式。3、多媒体课件辅助教学法。利用课件展示文章脉络，帮助学生梳理文脉，体会本文论点鲜明，论证严密的特点。<br/>
	    五、说教学程序：<br/>
	    1、课程以战国地图以及问鼎中原、群雄逐鹿图导入，探讨对战国历史的不同理解。2、对前课的重点虚实词进行复习。3、基于本学期的绝大数课程都与古典诗词和古文有关，因为设计了一个作业反馈环节，让学生来做古典诗词。探讨不同学生对于战国历史的不同理解。4、在此基础之上，确立《六国论》的文体，研究古典证论散文的特点，抛出4个问题，让学生讨论六国论的写作手法及内容理解。5、分析这篇文章的写作手法与论证方式，探讨其对议论文写作的指导意义。6、以自题绝句结课。</p>
<p>
	    （三）张岩：汇报课《氓》总结<br/>
	    问题：<br/>
	    一、教学任务完成情况：<br/>
	    因第一课时重点文言字词强调不到位，在第二课时检查复习情况时不够顺畅，占用时间略多，影响教学进度。<br/>
	    二、教学过程：<br/>
	    1、学生诵读诗歌时，有些读音错误没能及时纠正。2、讨论过程中任务分配不明确，可指定学生汇总小组讨论结果。3、有些地方思路不清晰，有重复讲解的地方，特别是对女主人公婚姻悲剧原因的分析，略显重复。4、对整个时代背景的把握不十分准确，对学生提出的具有当代物质色彩的某些思想应及时加以纠正，加强情感观及价值观的培养。<br/>
	    三、从师素质：<br/>
	    口头禅过多；个别环节衔接不够流畅；ppt和教学过程相互配合不够协调。<br/>
	    改进方向：增加自信，提升课堂驾驭能力；注意ppt与课堂进程的衔接；多阅读，增加文学积淀，以更好地解读、挖掘文本。</p>
<p>
	    二、组内教师评课汇总<br/>
	    1、耿英年：<br/>
	    （1）教学环节设计完整。包括已学知识的复习、交代学习目标明确课堂内容，重点内容的探讨研究，课外拓展，总结，作业布置；<br/>
	    （2）教学方法灵活多样。包括提问、讨论、学生发言质疑、教师拓展总结，体现了以学生为主体，教师为主导的教学理念；<br/>
	    （3）在重点难点的选择、突破上有特点。重点研究探讨了贯穿全文的情感，以“喜”“悲”两种情感为主，着重探讨了“悲”，并以“一个书斋、两种情感、三世变迁、四个女人”为切入点，深入探讨；<br/>
	    （4）教师素养深厚。能够适时调节课堂、升华学生的认识，提升课堂思维含量；<br/>
	    （5）不足：<br/>
	    ①已学知识的复习，从学生回答情况来看，学生对上节课的内容掌握不扎实，回答准确性、熟练性不够，不应该是特快班学生应有的水平。说明平常的训练、要求还要落到实处，语文课既要提升素养，又要让学生在考试中取得好成绩；<br/>
	    ②课外拓展内容超时3分钟，拖堂了。<br/>
	    ③学生发言气场不够。语文课应该在平日的课堂训练里义不容辞的承担起培养的责任，当然，其他科同样有这样的责任。<br/>
	    2、王丹：<br/>
	    （1）亮点频频：<br/>
	    ①作为第二课时，文章的字面障碍已经扫除，因此老师适时布置了学生对六国灭亡原因的探讨，并形成文字稿，总题目为《你眼中的战国》。课堂上，学生发言侃侃而谈，有理有据，气场十足。并且循着“读-谈-评-析”的思路，让全班学生讨论学生的文章，让一个人的认识让大家去分析、共享。闪光点体现在学生身上，外行人看到的是学生素质好，实际上反映的是教师大量的课后付出和长期、系统的训练；<br/>
	    ②文言教学很好的和写作相结合。《六国论》论证严密，层次清晰，论证方法多种多样，语言流畅，老师抓住本文特色，引导学生对写作技巧进行研习，这也是文言教学的一个新尝试。<br/>
	    （2）值得商榷之处：<br/>
	    ①句子翻译练习的精确性有待加强，要严格按照“字字有落实，不随意增删内容”的要求加强训练；<br/>
	    ②PPT字迹稍小，后排同学辨识有些困难。<br/>
	    3、张岩：<br/>
	    （1）教学环节完整、清晰、流畅。导入-交代教学目标-课堂练习-课堂探究-小结-课外拓展-作业布置，一气呵成；<br/>
	    （2）具备了很好的教师素养，在“导入”“拓展”环节，引入了大量的与文章相关的作品，丰富了文章内容，开阔了学生视野，提升了学生的思维含量，陶冶了学生的情感，已经具备了一名高水平教师所必须具备的潜质；<br/>
	    （3）教学方法多种多样并能熟练掌握。朗读、提问、讨论、探究、评析，驾轻就熟。<br/>
	    （4）重点难点把握准确、处理得当。课堂以“分析男女主人公的性格”为主线，带动对人物形象、性格、命运、悲剧的必然性的分析。在处理重点难点过程中，注意设置有难易度的问题，由浅入深，由表及里，逐次解决。<br/>
	    （5）教态落落大方，举止得体，作为一个刚刚参加工作四个月的年轻教师，难能可贵。<br/>
	    （6）不足：<br/>
	    ①PPT与教学进度配合不够默契，随着时间的增加，这一问题可迎刃而解。<br/>
	    ②自信稍显不够。</p>
<p>
	    三、精诚合作、团结协教<br/>
	    在三位老师准备上课的过程中，组内同仁能够主动帮助，查找材料、设计思路、试讲点评、调整修改……大家心往一处想，劲往一处使，尤其是各备课组长和一些老教师更是起到了传、帮、带的作用。<br/>
	    上完课后，上课的老师能在第一时间主动找其他教师点评，大家都能开诚布公说出自己对课的看法，指出优点的同时，又中肯地提出意见。<br/>
	    教研组内及时安排时间，组织大家对三位教师的课进行集体点评，做到“奇课共欣赏，疑义相与析”，在交流、讨论、点评过程中，思想得到碰撞、升华，教学的艺术在这样的过程中得到提升。<br/>
	    一个人的智慧总是有限的，集体的智慧是无穷的。本次上课的全过程体现了精诚合作、团结协教的优良作风，这也应该是语文组一笔宝贵的精神财富。</p>
