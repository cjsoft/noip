《天空没有多余的星星》——黄启成校长在2012届毕业典礼上的致辞



<table width="100%">
<tr>
<td><div id="contentMidPicAD" style="float:right; clear:both; top:0; vertical-align:top;"></div>
<p>
	 </p>
<p>
<strong>    尊敬的各位老师、家长，亲爱的同学们：<br/>
<br/>
	    大家好！　　<br/>
<br/>
	    同学们，时间过得真快！转眼，毕业已猛然站在你眼前。带着青春的欢乐、骄傲、活力，还有些许的伤感。这不是你的第一次毕业，肯定不会是最后一次。无论你现在是何种心情，这一天还是来了；居然来了；终于来了！这注定是你人生中最重要的一刻，一场最生动的毕业教育。<br/>
<br/>
	    首先，我代表24中，欣慰地也真诚地祝贺你们——24中最年轻的校友——光荣毕业！同时感谢同学们给予老师和学校的无数次深深的感动和喜悦，感谢同学们让家长们拥有更多的幸福和期待。我也向所有为了同学们的成长而呕心沥血、辛勤耕耘的老师们致以崇高的敬意！向多年以来给予学校极大的信任和支持的所有毕业生家长表示衷心的感谢！另外，我还要感谢刚刚讲话的家长代表、感谢刘奕同学，感谢同学们捐赠给学校的那份宝贵的基金。<br/>
<br/>
	    同学们，三年的高中生活，你们不仅学业优秀，你们还以热情、真诚、睿智、魄力将年轻人的梦想描绘的五彩斑斓、勾画的流光溢彩，通过你们的表现，充分彰显了24中求真尚美、尊重个性、全面发展的理念。<br/>
<br/>
	    三年来，我们努力想使你们感到快乐，我们希望你们在人性光辉的照耀下快乐健康成长！这是我的同事们超越时限忘我工作、家长含辛茹苦无怨无悔关怀你们的动力源泉。尽管三年的学校生活可能不那么事事如意（比如分班，可能成为有的同学永远的痛，还有食堂的饭菜、“校车”有时的迟到和甩站、时有发生的“不公正的待遇”，如此等等），但我仍认为你们得到了良好的教育。良好的教育的标准不是你们一定要考进清北、港大，还有其他世界级名校，而是通过学校教育使你们的身体变得强壮、思想变得深邃、情感变得丰富，以及拥有乐观豁达、积极向上的人生态度，友好互助、团队合作的人文情怀。 <br/>
<br/>
	    同学们，过去我说过，真正的人生从大学开始，今天我换个说法，进入大学才是真正踏入社会。今天的毕业典礼，也是纪念同学们即将告别的高中生活。但今天的话题我不打算太多关注校园，几句我要与大家分享：<br/>
<br/>
	    第一，人们常说“今天是一个充满生机和挑战的时代，是社会转型、社会道德共识重建的时代。换个说法，今天是一个充满诱惑、外在规范特别是制约不够的年代。这不仅挑战法治，更挑战一个人的德性、操守和判断力”。<br/>
<br/>
	    为此，作为你们的校长，我想说：你们要优秀！不单单是做一名优秀的学生，更要做一个优秀的人。作为24中人，理当终生进取、追求卓越、服务社会，报效国家，这是我们共同热爱着的这所学校精神的重要内涵。<br/>
<br/>
	    作为你们的长辈，我想说：希望你们幸福。幸福是一种非常强烈和深刻的快乐，会让人由衷的感觉到人生是多么美好，而决定一个人感受幸福能力的，是内心世界的丰富、敏感和活跃。我不担心你们会忘记追求优秀，我担心你们忘记热爱生活，忘记优秀的目的是幸福。<br/>
<br/>
	    作为你们的朋友，我还想说：你们要挺胸直立，骄傲而无所畏惧。这是青年人最美、最吸引人的风姿。希望你们能发扬并传承这种精神，去梦想，去创造，去失败，去成功。<br/>
<br/>
	    第二，如果你“有侥幸心理，把握不了自己，忘乎所以，随波逐流，一不小心也可能混迹于成功人士，但请你记住鲁迅先生的话，大意是，如果你真能折腾，真会忽悠，也会小有斩获；但想凭此成就大事，自古以来，门儿都没有”。<br/>
<br/>
	    为此，作为校长、长辈和朋友，我还要最后再重复一遍我们经常说的几句至理名言：<br/>
<br/>
	    苏格拉底说，美德即智慧。但丁说，道德可以弥补智慧的缺陷，而智慧永远弥补不了道德的缺陷。<br/>
<br/>
	    孔子说：志于道，据于德，依于仁，游于艺。李政道说，科学和艺术是一枚硬币的两面，二者缺一不可。<br/>
<br/>
	    培根说，读史使人明智，读诗使人灵秀，演算使人精密，科学使人深刻，伦理学使人庄重，逻辑修辞学使人善辩。凡有所学，皆成性格。<br/>
<br/>
	    韩愈说，师不必贤于弟子，弟子不必不如师。亚里士多德说，吾爱吾师，吾更爱真理。<br/>
<br/>
	    第三，天空没有多余的星星，每个人都有极大的潜能。当独自面对自己或永恒时，你要更看重做事，努力做成事，而不是太计较所谓的公正回报，太多关注关心别人对你的看法。<br/>
<br/>
	    今天，借此机会，讲一个小故事，关于戴尔•泰勒牧师和比尔•盖茨的故事。（见《读者》2005年第22期《尽力而为还不够》）这个故事给我们的启示：凡事仅仅做到尽力而为还不够，谁要想创造奇迹，必须竭尽全力！ <br/>
<br/>
	    同学们，替代公式化毕业致辞的并非只是“柔软地想起这个校园”。我相信你们，我不会忘记你们。我也相信，你们不会忘记曾让你们欣喜若狂也曾让你们焦虑不安，让你们由衷赞美过也曾让你愤愤诅咒过的24中。<br/>
<br/>
	    面对今天的中国和世界，我们必须超越昨天和自己；我们拥抱，却不止步于感性和温情！而且，我相信，无论如何，这里都会是你心中最好的校园，留下了你的一段刻骨铭心；种种失意，哪怕是失望，经过时光打磨，都会成为你回忆中的美好和亲切。其实，记住一点也就够了：6月1日晚8点，你们的学弟学妹们代表所有24中人，通过灯光传达给你们的深情祝福！<br/>
<br/>
	    同学们，你们把青春中最美好的一段时光给了24中，24中也因你们而美丽、而年轻。年年岁岁人不同，岁岁年年“话”相似。未来其实并不遥远，当一个个未来变成“此刻”时 ，“此刻”不就是你曾经眺望过的一个未来？！ <br/>
<br/>
	    谢谢大家！</strong></p>

<strong>二〇一二年六月二十一日</strong></p>
