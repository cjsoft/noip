2、录取报到地点：学生处<br/>
<br/>
3、入学考试要求提前十分钟到考场，<u>凭准考证及护照进入考场，证件不全者不允许进入考场</u>。<br/>
<br/>
4、考试科目及内容：<br/>
<br/>
（1）语文能力测试：满分150分，<u>包含作文</u>。<br/>
（2）数学：满分150分，以中考考纲为标准。<br/>
（3）英语：满分140分，以中考考纲为标准，<u>没有听力</u>。<br/>
（4）物理、化学：满分160分，以中考考纲为标准。<br/>
<br/>
5、录取结果我们会用电话通知，请家长和孩子保持电话畅通，否则后果自负。<br/>
<br/>
6、被录取的考生请携带以下资料来我校报到。<br/>
<br/>
（1）父母在我市投资或正式就职并在我市居住的外籍学生，需要提供：<br/>
<br/>
①父母护照及职业签证                <br/>
②父母及学生在连居住证明原件及复印件一份（A4纸复印）<br/>
③父母在连投资证明或就职证明        <br/>
④学生护照原件及复印件一份（A4纸复印）<br/>
⑤中国公证机关签发的亲子关系公证文书<br/>
⑥《大连市中小学外籍学生入学申请表》一式四份（可从所在区教育局获取或通过“大连市教育局网站-网上办公-表格下载”下载，用A4纸打印）。<br/>
<br/>
（2）以留学为目的到我市中小学就学的外籍学生，如父母在连常住，需要提供：<br/>
<br/>
①父母护照                          <br/>
②学生护照原件及复印件一份（A4纸复印）<br/>
③父母及学生在连居住证明原件及复印件一份（A4纸复印）<br/>
④中国公证机关签发的亲子关系公证文书<br/>
⑤《大连市中小学外籍学生入学申请表》一式四份（可从所在区教育局获取或通过“大连市教育局网站-网上办公-表格下载”下载，用A4纸打印）。<br/>
<br/>
（3）<u>如父母不在连常住委托在连常住人员作为监护人，需要提供</u>：<br/>
<br/>
①学生护照            <br/>
②学生和委托监护人在连居住证明原件及复印件一份（A4纸复印）<br/>
③中国公证机关签发的委托监护公证文书<br/>
④委托监护人的身份证明原件及复印件一份（A4纸复印）<br/>
⑤《大连市中小学外籍学生入学申请表》一式四份（可从所在区教育局获取或通过“大连市教育局网站-网上办公-表格下载”下载，用A4纸打印）。<br/>
