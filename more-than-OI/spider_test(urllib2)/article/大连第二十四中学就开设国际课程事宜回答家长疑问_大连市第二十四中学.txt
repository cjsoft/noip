大连第二十四中学就开设国际课程事宜回答家长疑问



<table width="100%">
<tr>
<td><div id="contentMidPicAD" style="float:right; clear:both; top:0; vertical-align:top;"></div>
<p>
	 </p>
<p>
<strong>    经剑桥大学国际考试委员会授权，辽宁省教育厅批准，大连市第二十四中学与英国布罗德格林国际学校合作，自去年起开设剑桥国际高中课程。这是我市唯一经批准在大连市教育局所属公办普通高中开设的此类课程。面对众多家长在招生阶段的关注和咨询，学校就几个问题做一解答。</strong></p>
<p>
<br/>
<br/>
<strong>    问：剑桥国际高中课程班提供哪些学习科目？<br/>
<br/>
	    答：剑桥国际高中课程源于英国全民普及的高中课程，被誉为英国“金牌课程”及“世界高等教育的通行证”，所开设的课程有英语语言、数学、进阶数学、物理、化学、商业学、经济学等。<br/>
<br/>
	    国际班学生在学习上述课程的同时，学习国内普通高中课程。</strong></p>
<p>
<br/>
<br/>
<strong>    问：剑桥国际高中课程班如何进行教学管理？<br/>
<br/>
	    答：剑桥国际高中课程班由大连市第二十四中学全面管理，由剑桥学术委员会进行教学监督。<br/>
<br/>
	    国际课程采用小班化授课方式，全部由具备国际教师资格、拥有相应教学经验的英语国家的8位外教直接授课。国内课程则由我校选派优秀教师负责教授。<br/>
<br/>
	    学生周一至周五全日制上课。课后给学生留出充裕的时间参加课外活动，或准备雅思、托福及SAT考试。</strong></p>
<p>
<br/>
<br/>
<strong>    问：学生毕业后获得什么文凭？<br/>
<br/>
	    答：对学习期满达到毕业要求，并考核合格的学生，颁发中国辽宁省普通高中毕业证书和英国剑桥国际高中证书。英国剑桥国际高中证书作为评估学生学习能力的权威，已得到世界上150多个国家1100多所大学的认可。</strong></p>
<p>
<br/>
<br/>
<strong>    问：毕业生有机会申请什么样的大学？<br/>
<br/>
	    答：学生经过三年的学习，通过剑桥大学国际考试委员会的全球统一考试后，可以根据自己的成绩和意愿在全球范围内选择申请美国、英国、澳大利亚、加拿大、新西兰、新加坡等国家英语授课的大学，包括哈佛、普林斯顿、耶鲁、剑桥、牛津等世界顶级大学。根据对在读学生的摸底调查，80%以上的学生拟选择到美国一流大学完成大学学业。<br/>
<br/>
	    学校应学生要求，给予学生一对一个性化升学指导，并协助办理升学相关事宜。</strong></p>
<p>

<strong>     *剑桥国际课程本年度招生人数及入学要求请参照大连市第二十四中网站的招生简章</strong></p>
