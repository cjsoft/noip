“他们，就是明天的希望”——爱心社辅导站活动

<td><div id="contentMidPicAD" style="float:right; clear:both; top:0; vertical-align:top;"></div>
            在城市的尽头,没有繁华的街市,闪亮的霓虹灯;在城市的尽头,只有破旧的棚户区和饱经生活风霜的生命;在城市的尽头,有这样一群人。<br/>
  <br/>
     让我怎样称呼他们？外来务工人员子女？农民子弟？抑或是农民工二代？不，我不想用这些冰冷的名字称呼他们，我多想叫着他们带着泥土气的乳名，拉着他们的小手，走进他们的生活……<br/>
  <br/>
     他们从小生长在故乡的青山绿水中,纯净的灵魂在田野里抽穗拔节。在山野的风中，他们奔跑着，憧憬着。风从田野中吹过，吹进了城市，为了生计，为了未来，他们跟从父母来到了城市，在城市的尽头扎下了根。<br/>
  <br/>
     于是习惯了青山绿水的双眸第一次触碰到了高楼大厦、车水马龙。他们不知道怎样穿过六车道的马路，小小的手指怎么也数不清写字楼的层数。他们背起书包，小心翼翼地融入城市的生活。可是却在城市人的眼中，第一次明白了户口本与暂住证的区别。他们都是父母心头的宝贝却过早地承担了不属于这个年龄的负担。<br/>
 <br/>
     然而在他们日渐长高的身体上，清脆的声音中，我看到了他们的成长。会争先恐后地举手朗读对话，会异口同声地大声背诵出beautiful, young，会在课程结束时向我们鞠躬：“欢迎老师再来！”。<br/>
 <br/>
     记得第一次进入简陋的教室，看到孩子们一张张殷勤期盼的笑脸，心中略些紧张与不安。其中一个孩子把我拉到她的身边，费劲地用两只小手抬起一把凳子：“老师您坐！”。记得坐上返程的车，一个小男子汉趴在车窗前，不停地向我们摆着手。那一刻猛然觉得他们是一群值得爱的孩子。<br/>
 <br/>
     记得他们稚气的宣言：“我们的学校很小，但我们的成绩不差！”“我的梦想是长大了做一名大科学家！”“我将来一定会为父母争光！”他们逐渐成熟，告别昨日的羞怯，开始迎接新的一天。<br/>
 <br/>
     虽然他们还在为不多的学费而苦恼，虽然他们还在微弱的灯光底下读着书，虽然还有好多制度还不够完善……虽然还有好多个虽然，但是，只有只要有一个但是就足够了，已经有包括我们的好多视线转向他们，他们正在茁壮地成长。<br/>
 <br/>
     我想我们的活动我会一直坚持下去，不仅仅是为了他们，更是为了明天的希望。<br/>
<br/>
<br/>
     太阳从地平线上升起，照亮了城市的尽头，照亮他们的生活。<br/>
 <br/>
     他们，终将会成为我们。<br/>
 <br/>
     他们，就是明天的希望！
