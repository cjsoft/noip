我国教改试点确定建设现代大学制度等十大任务

      国务院办公厅日前印发了《关于开展国家教育体制改革试点的通知》，确定了国家教育体制改革试点的主要任务和试点单位，各地各校申报的国家级试点项目即将全面启动实施。<br/>
<br/>
<strong>　　超</strong><strong>500</strong><strong>项目参与第一轮申报</strong><br/>
<br/>
　　国家教育体制改革领导小组办公室负责人表示，我国教育改革已进入“深水区”，面临着很多前所未有的新情况、新问题、新矛盾，改革的难度大、压力大，矛盾也比较多，许多问题没有现成答案，需要在实践中探索。<br/>
<br/>
　　据了解，从今年年初开始，教育规划纲要工作小组办公室就结合规划纲要提出的重大改革，研究确定了近期启动实施的重点任务，并将这些重点任务公开发布，请愿意先行先试的地区和学校，选择一项或几项申报国家改革试点，按照“自愿申报、专家评审、协商论证、综合平衡、统一部署”的原则，制定国家教育体制改革试点总体方案。在第一轮申报的时候，共有500多个改革项目，各地各校改革的积极性很高。<br/>
<br/>
<strong>　　领导小组由</strong><strong>20</strong><strong>个部门组成</strong><br/>
<br/>
　　该负责人指出，为加强对全国教育体制改革工作的组织领导，国务院成立了国家教育体制改革领导小组，由20个部门组成，主要职责是审议教育改革发展的重大方针和政策措施，研究部署、指导实施教育体制改革工作，统筹协调教育改革发展中的重大问题。国家教育体制改革试点工作由国家教育体制改革领导小组组织领导，省级人民政府和国家有关部门组织实施。同时，为加强对教育体制改革试点的指导，国家教育咨询委员会已于11月中旬成立。各地各校的试点项目实施方案，都要经咨询委员会评审、完成备案程序后，才能启动实施。<br/>
<br/>
　　该负责人称，改革试点启动后，省级人民政府和国家有关部门要建立督促检查机制，按照试点实施的计划进度，开展跟踪调研，及时了解情况，定期进行评估。对于实施中可能存在的风险因素，要深入分析和系统评估，做好预案，积极化解。对于实施过程中发现的问题，要认真研究，及时妥善处理，避免出现大的偏差。国家教育体制改革领导小组将根据试点进展情况对试点项目进行动态调整和补充。<br/>
<br/>
<strong>　　改革试点十大任务</strong><br/>
<br/>
　　1、建立健全体制机制，加快学前教育发展<br/>
<br/>
　　2、推进义务教育均衡发展，多种途径解决择校问题<br/>
<br/>
　　3、推进素质教育，切实减轻中小学生课业负担<br/>
<br/>
　　4、改革职业教育办学模式，构建现代职业教育体系<br/>
<br/>
　　5、改革人才培养模式，提高高等教育人才培养质量<br/>
<br/>
　　6、改革高等教育管理方式，建设现代大学制度<br/>
<br/>
　　7、改革高等学校办学模式<br/>
<br/>
　　8、改善民办教育发展环境，深化办学体制改革<br/>
<br/>
　　9、健全教师管理制度，加强教师队伍建设<br/>
<br/>
　　10、完善投入机制提高保障水平<br/>
