<p>
<font color="#000000">        不知道什么时候开始，突然想成为一个作家的念头，写很多书，还要写一本关于自己的书！但是我知道，真正做一名作家写一本书谈何容易！虽然，我平时喜欢看书，看各种不同类型的书籍。但是我看书和其他人是不一样的，别人看书是认真思读书中内容，品内容之精华，而我看书书是“看”而不是看。我的看代表代表对一本书的钟情和喜欢，我喜欢看书并不是一味着看书中的内容，我的性情是：我喜欢书，我的生活中不能没有书的影子，就像一个人不能失去自己的恋人一样。有时候，我也会经常翻阅自己喜欢的书里面的内容，不过那也只是囫囵吞枣罢了。<br/>
	　　<br/>
	　　其实，我就是想摸一摸，翻一翻就行，就觉得很舒服已经很满足了，所以，我只感受到了物质上的存在，而忽略切遗忘了她实质中的真谛，以至于今天要写一本书为什么会那么困难；一句话为什么那么地不完整，那么的含蓄了。也许，我不是一个当作家的料吧？我只是一个喜欢“看”书而不是一个看书的人！<br/>
	　　<br/>
	　　每当我遇到某些不愉快的事或者心情低落甚至悲观成疾，那个时候，我多想用文字来表达我的思绪与心情，然后给别人去看，给更多人去分享，分享我的快乐与失落时候的悲观。然而，我却没有能那样做，一次又一次沦落在消极落寞的世界中徘徊。其实，有时候想想，也觉得很简单。我只要把自己内心所想的，想要说的话用语句表达清楚就行了。但是，当你真正投身去做这件事的时候就难了，因为，那是需要足够的耐心与毅力的！还有就是我该具备哪些关于写作的思路与心理！想要成为一名作家，更何况是一名好的作家。不是说只要会写一篇好的文章，然后把这篇文章些地很长，很长！以至于成一本书效果那么对的书的文字你就是一个作家了，那只不过是中学生写论文罢了。但是应该能评一个作文精神奖吧。我觉得想要写出一篇好的文章首先必须写好要写的题材，我要写什么？想好要写什么之后我又该怎样去写？写出来的文章会达到什么效果？有没有可读性？使读者一看就能明白文章的内容与我要表达的是一个什么样的意思，能得到大众的认可！写文章就是读者们去看；去分享；去讨论，然后去评论文章的好坏，得到与不足之处。因为，最忠实的读者才是你以后能写好写更多文章的向导！这就是大众化的认可与肯定！最后也是最重要的一点就是“真实性”不管我写什么样的文章，都要体现文章的主体，还有就是实际性，与与现实社会是否相结合。使人看后能从中明白一些事理或者感悟。我觉得，这就是写作中界中的真谛与精髓！但这只是我个人认可的观点，并不是大众所肯定的。<br/>
	　　<br/>
	　　我看过多许多名人作家的文学著作，其作风（写作风格）自然是不一样的。我个人偏爱鲁迅先生多一点，他的作品思想深邃，不仅为中国现代文化提供了取之不竭的精神资源，也为我今后是否能真正写好文章提供了初步与不朽的平台.....<br/>
	　　<br/>
	　　说到这，我还想特别提一下另外一个作家：美国著名的女作家——（海伦·凯勒）。<br/>
	　　<br/>
	　　在她的生涯中可谓是经历大风大浪的，我不能说在她的写作生涯。因为她不只是个著名的作家而且还是一个伟大的教育家。<br/>
	　　<br/>
	　　在她出生十八个月的时候，因病失去了视觉和听觉，成为残疾人。从七岁起，她在家庭教师的帮助下学习用手触摸识字，开口说话，与人交流。在自己的不懈努力与优秀恩师的帮助下，后以优秀成绩毕业于哈佛大学附属剑桥女子学校和拉德克利夫学院。在她的文学生涯中，主要作品有自传《我的生命故事》。但她留给我印象最深刻的一篇著作便要属《假如给我三天光明》这篇散文。<br/>
	　　<br/>
	　　在《假如给我三天光明》中，作者从一个残疾人的特殊角度，用心灵去感受，去触摸，去想像，去摸索大千世界，这给了我很深的启迪。<br/>
	　　<br/>
	　　在奇妙的设想中，作者“第一天”看到的是“那些以他们的慈爱，温情和友谊，使我的生命值得活下去的人”：“第二天”通过观看人类的艺术去探索人类的灵魂”：第三天“作者要到忙于事物的人们的地方去看看当前的日常世界。”文中，作者以含感情地为我们描绘了她想像中的人类历史；自然景物，绘画艺术以及各种生活场。作者对这些描绘地细致入微。使我在阅读的过程中产生一种错觉——感觉的她才是健全的。言为心声”，作者那些美好的设想恰恰充分展现了她那丰富多彩的内心世界，表现出了她对人类，自然的强烈的热爱，对生命的执着追求。<br/>
	　　<br/>
	　　当我以热中的好奇心看完这篇散文之后，感到特别地震惊与自卑心理！她，一个失去视觉和听觉，每天生活在没有声音没有光明的黑暗世界。竟能把自己看不到和听不见的的一切美好事物，绘声绘色地展现在了我眼前。而且比常人“看”得更透彻，“听”得比常人更广阔。这不得不使我会在读的过程中产生一种错觉，她是健全的，我则是假面的健全人！由此，这不得不让我感到非常地震惊和自卑！<br/>
	　　<br/>
	　　在读完这篇文章之后我也深深地领悟到了，要想写一篇好的文章不仅需要耐力与毅力的同时，而且还要用心去感受去体会更要用心去摸索去探索世间一切美好由衷的事物.....</font></p>
