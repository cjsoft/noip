<strong>教什么知识比怎么教知识重要</strong></div>
<div style="text-align: center">
	张坪    辽宁省大连市第24中学   </div>
 <br/>
      课程目标的基础目标是知识目标，离开知识的传授，就无法进行方法、能力、价值观的培育，可见知识教学的重要。人们习惯上认为，教师的教学就是教知识，可是，什么知识更重要?什么知识更有价值？哪些知识是学生终身发展所必须具备的？这是一个非常重要的问题，它直接影响学与教有效性的问题，是课程与教学的理论和实践的基本问题，也是当下每一位教师必须考虑而又缺乏考虑的一个问题。<br/>
<br/>
<strong>      一、</strong><strong>关于知识概念的某些观点</strong><br/>
     知识概念的几种观点<strong>：知识是人们改造世界的实践中的认识和经验的总结。知识是人类对客观事物及其规律的反映，是人类在长期社会实践中积累起来的认识成果。科学是经过实践检验具有严密的逻辑论证，反映客观事物本质和规律的知识体系。</strong>科学知识是具有一定的内在联系和逻辑系统，从中选择最基础的东西传授给学生，这就是基础知识基本理论的教学，这是普通教学的基本任务之一，也是能力方法教育与情感态度价值观，教育基础或载体。<br/>
     苏联著名教育家巴班斯基说：“知识是经过实践检验的对现实认识的结果，是现实在人思维中的正确反映。知识反映人们在社会历史实践过程中所积累的概括性经验。科学知识最正确地反映客观世界，并且随着客观世界的发展而不断变化和完善。”<br/>
     说到知识，人类的知识可分为两大门类：一是自然知识，它是生产实践和科学实验的经验总结，另一门是社会知识，它是人类社会实践的总结，自然科学和社会科学就是这两大门类的知识的结晶，哲学则是自然知识与社会知识，概括与总结。关于知识的类别问题，哲学家有过多种划分法。按照知识对现实反映的水平，可分为经验的知识和理论的知识；经验的知识是观察和描述性的，理论的知识是概括性的或构成假说的。按照客观对象和研究方法的不同，可分为自然科学知识和人文科学知识，描述的知识和规范的知识，分析的知识和综合的知识，内容方面的知识和形式方面的知识。除科学知识外，还有日常生活知识和技术知识等。<br/>
<br/>
<strong>     二、什么是知识？</strong><br/>
　<br/>
     汉语词义 1.“知识”的“知”，字从矢从口，矢亦声。“矢”指“射箭”，“口”指“说话”。“矢”与“口”联合起来表示“说话像射箭，说对话像箭中靶心”。本义：说的很准（一语中的）。 “不知”或“未知”就是指话没有说准，就好像射箭没有击中靶心。　2. “知识”的“识”，繁体写作“识”字从言从戠，戠亦声。“戠”字从音从戈，本指古代军队的方阵操练。“戈”指参加操演的军人及其武器。随着教官的一连串指令声，军阵会出现整体前进或后退、整体左移或右移、横排队“识（识）”的本义、引申义。本义：用语言描述图案的形状和细节。 引申义：区别、辨别。　　3.“知识”一词的汉语本来词义：指学术，文化或学问。<br/>
     在汉语中，“知”和“识”出于同一语族。如《玉篇》曰：“知，识也。”笼统地说，“知”和“识”都指人对外在事物的反映，但二者对外在事物的反映方式则不同。《集韵》将“知”解释为“觉。”“觉”就是感觉，意在说“知”是由各种外界的刺激作用于人的感觉器官后，再由人脑反映出来的意象。<br/>
<br/>
<strong>     三、教育心理学上的“知识”概念与知识分类</strong><br/>
<br/>
     个体通过与环境相互作用后获得的信息及其组织，在心理学上称之为知识。　知识的分类　按现代认知心理学的理解，知识有广义与狭义之分。广义的知识可以分为两类，即陈述性知识、程序性知识。　<br/>
<p>
	1、陈述性知识　陈述性知识是描述客观事物的特点及关系的知识，也称为描述性知识。陈述性知识主要包括三种不同水平：符号表征、概念、命题。　符号表征是最简单的陈述性知识。所谓符号表征就指代表一定事物的符号。例如学生所学习的英语单词的词形、物理公式中的符号、化学元素的符号、历史学习中的图画、旗帜、器皿、地图上的地理图标等都识符号表征。概念是对一类事物本质特征的反映，是较为复杂的陈述性知识。　例如历史学习中的概念包括“鸦片战争”、“五四运动”等历史事件类概念；“华盛顿”“克伦威尔”等历史人物类概念；《权利法案》、美国1787年宪法等典章文献类概念等等。命题是对事物之间关系的陈述，是最复杂的陈述性知识。例如历史学习中的“1787年美国宪法是近代世界第一部成文宪法”、“辛亥革命是比较完整意义上的资产阶级民主革命”、“英法资产阶级革命一定程度上代表了世界历史的发展趋势”等命题。<br/>
<br/>
	2、程序性知识　程序性知识是一套关于办事的操作步骤和过程的知识，也称操作性知识。这类知识主要用来解决“做什么”和“如何做”的问题，可用来进行操作和实践。例如历史学习中的“如何评价历史人物”、“如何解析历史文献资料”等都是程序性知识。<br/>
	 <br/>
	     可见，陈述性知识与程序性知识的区别主要有以下几个方面：第一，陈述性知识是“是什么”的知识，以命题及其命题网络来表征；程序性知识是“怎样做”的知识，以产生式来表征。第二，陈述性知识是一种静态的知识；而程序性知识是一种动态的知识，它的激活是信息的变形和操作。第三，陈述性知识激活，需要学习者对有关事实进行再认或再现；而程序性知识激活的速度很快，体现相对稳定的思维定势及自动化了的信息变形的活动。在人们的社会活动中，两类知识是结合在一起的；在学生的学习过程中，最初都以陈述性知识的形式来获得，只是在大量练习之后程序性知识才具有了自动化的特点。<br/>
<br/>
<strong>     四、</strong><strong>知识管理中的知识</strong><br/>
<br/>
	     从信息论角度看，知识是这样一个过程，是信息的管理和分类过程，让信息从庞大无序到分类有序，各取所需。这就是一个知识管理的过程，也是一个让信息价值升华的过程。　结论，知识就是一切人类总结归纳，并认为正确真实，可以指导解决实践问题的<a href="http://baike.baidu.com/view/686797.htm">观点</a>、经验、程序等信息。 　　知识作为一种特殊的信息，它具备了更多的附加特征，也就是说，某一种信息如果越多增加这种特征的烙印，就越接近知识。 　　<br/>
	     知识特征：隐性特征，知识具备较强的隐蔽性，需要进行归纳、总结、提炼。<br/>
	     行动导向特征，知识能够直接推动人的决策和行为，加速行动过程。动态特征知识不断更新和修正。主观特征，每个人对知识的理解，都会加入自己的主观意愿。可复制/转移，知识可以被复制和转移，可重复利用。延展生长特征，知识在应用、交流的过程中，被不断丰富和拓展。资本特征，知识就是金钱。倍增特征，知识经过传播不会减少，而会产生倍增效应。一个苹果两人分享，一人只有半个，一个知识两人分享，就至少有两条。熟练特征，知识运用越熟练，有效性越高。情境特征，知识必须在规定的情景下起作用，人类选择知识一般都会进行情境对比。心智接受特征，知识必须经过人的心智内化，真正理解，才能被准确运用。结果导向特征，知识不但加速过程，也导向一个可预期的结果。权力特征，掌握知识的人，即便不在职务高位，也拥有一定的隐性权力。生命特征，知识是有产生和实效的过程，有生命长短，不是永久有效的。<br/>
	     隐性知识是迈克尔·波兰尼（Michael Polanyi）在1958年从哲学领域提出的概念。他在对人类知识的哪些方面依赖于信仰的考查中，偶然地发现这样一个事实，即这种信仰的因素是知识的隐性部分所固有的。波兰尼认为：“人类的知识有两种。通常被描述为知识的，即以书面文字、图表和数学公式加以表述的，只是一种类型的知识。而未被表述的知识，像我们在做某事的行动中所拥有的知识，是另一种知识。”他把前者称为显性知识，而将后者称为隐性知识，按照波兰尼的理解，显性知识是能够被人类以一定符码系统（最典型的是语言，也包括数学公式、各类图表、盲文、手势语、旗语等诸种符号形式）加以完整表述的知识。隐性知识和显性知识相对，是指那种我们知道但难以言表的知识人类进入<a href="http://baike.baidu.com/view/487.htm">知识经济</a>的时代意义：　日本<a href="http://baike.baidu.com/view/2057.htm">知识管理</a>专家野中郁次郎（Ikujiro Nonaka）提出了显性知识和隐性知识相互转换的SECI过程： <a href="http://baike.baidu.com/view/777806.htm">野中郁次郎</a>（Nonaka）认为知识是一种被确认的<a href="http://baike.baidu.com/view/48078.htm">信念</a>，通过知识持有者和接收者的信念模式和约束来创造、组织和传递，在传递知识的同时也传递着一套文化系统。知识是从不相关或相关的信息中变化、重构、创造而得到的，其<a href="http://baike.baidu.com/view/78306.htm">内涵</a>比数据、信息要更广、更深、更丰富。此观点强调知识与背景，以及知识与信息的关系。<br/>
	     教学启示：知识需要生成与建构，知识教学观念必须转变。<br/>
