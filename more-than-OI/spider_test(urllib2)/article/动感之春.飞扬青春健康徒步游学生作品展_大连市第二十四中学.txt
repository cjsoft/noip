<strong>春游有感</strong><br/>
 <br/>
一年五班       刘歆子建 <br/>
 <br/>
从小学到高中，经历的春游不下十次，但不是去游乐园，就是去博物馆，失去了春游原本赏春他踏青的价值。今年允许独自散布还是第一次遇到，与以往相比，这种方式必定会引起更大的兴趣。<br/>
看看班级里的其他同学，也都与我差不多，备足了食物和水源，也有一些应急的物品以及娱乐设施。因为这对于我们每个人来说都是一个难得的机会。经历了近两个小时的徒步旅行，我们来到棒棰岛海滨。春光普照，海风拂面，沙滩上的同学正以各种方式享受春天带来的无比欢乐！<br/>
然而，在并不遥远的过去，春游又是什么形式呢？<br/>
四十、五十年代，上初中的祖母春游的目的地便是学校旁的一座山，活动很简单，在老师的带领下，共同爬山，同时欣赏沿途风景。但当时他们不叫春游，叫远足。到了中午，他们就以从家里带来的煮高粱米饭充饥，如果家中有条件，或者父母奖励孩子，还会带一只咸鸭蛋。可想而知，当时生活十分贫穷，但一次远足活动仍然是当时学生很长时间所期盼的活动。<br/>
七十年代，上初中的父母也曾随学校一同踏青。当时境况就改善了不少，父亲时代他们的食物是面包，几片香肠，甚至一两瓶汽水。春游也改成了到公园里观赏春景，实现了真正的享受！<br/>
又过了二十多年，如今的我们生活水平已有了很大程度的提高，春游也成为了生活中浓墨重彩的一笔。这至少就是社会发展的一种体现。<br/>
我们不禁会想，再过二十多年，我们的生活会有多大的变化？歌词里写的好：<br/>
“再过二十年，我们来相会，荡起小船儿，暖风轻轻吹。花儿香，鸟儿鸣，春光惹人醉，欢歌笑语绕着彩云飞。啊，亲爱的朋友们，美好的春光属于谁？属于我，属于你，属于我们跨世纪的新一辈。”<br/>

<strong>棒棰岛之恋</strong><br/>
一年五班        黄盛      <br/>
从公交车走下的一刻，扑面而来的青草的香味儿和满目的绿色几乎将我淹没。漫山遍野的绿色——一种透明的泛着嫩黄色的青草绿，瞬间濯去一切凡尘，让我的心灵也变得水灵灵的。不远处一条路通向山上，我们的徒步旅行将从那里开始。<br/>
春日的阳光那样明媚灿烂，暖暖的，柔柔的。风儿轻轻拂过，捎来一些凉爽和花香。浸在这样的阳光中，沐浴在幽幽的花香里，我的记忆之门也打开了。这条山路我是走过很多次的。小时候家离棒棰岛很近，晚饭过后便常来这条路上散步。暮春时节空气中浮动着潮湿的水气，那再熟悉不过的草味儿更加浓郁而又沁人心脾。远远的云端太阳像醉了的抽象派画家，将大片红的、橙的燃料泼洒在棒棰岛的山野间、小路上，一切都徜徉在温暖和静谧之中。偶尔有归巢的鸟儿划过天际，留下一声惘惘然的轻鸣；偶尔在道旁看到忙碌的蚁群，排着整齐的队在草叶间穿行。同样温柔的晚风习习而吹，让儿时的我沉醉，又让我在记忆中不断回味再回味。<br/>
但上午的景色又与傍晚时分不同。在如此明亮耀眼的阳光里，一切都能看的十分真切，那绿意也十分明澈，不像记忆中那样笼在一层恬静的晚霞中。每一片草叶、树叶都绿得纯粹、阳光，每一片花瓣都嫩得惹人怜爱。实在让我不思去碰触。在高处能外望见海，反射着银光万点的海似乎仍睡着，与蓝天融为了一体。<br/>
中午的时候我们到了海边。棒棰岛就在眼前了。这个名字很形象，阳光下的棒棰岛真有几分像放在一块绸布上的棒棰。海边又换了一种味道，这是一种咸咸的海水味，一种对大连人再亲切不过的海的气息。海滩上密密地撒落着被海水打磨得圆滑晶亮的小石头，在那样完美的椭圆、那样奇妙的纹理和被海水抚摸过后玲珑的的色泽，以及那种对着日光几乎能看透的晶莹，让人不得不佩服大自然的鬼斧神工。一排排雪白的浪涌进，打碎，如此不疲，又让人联想起人生和岁月……<br/>
当你坐在沙滩上凝视大海时，你似乎感到你在前进，你乘着一艘船破开浪花向棒棰岛驶去，然而它永远和你有一段距离，不远又不近。这是一种怎样奇异的感觉啊，于是我不禁又沉醉了。<br/>
美丽的时光永远短暂，然而一旦被记忆定格，它将成为永恒。棒棰岛，它几乎成为大连旅游的代名词，它的绿，它的香，它的温柔，它的亲切让所有感受过它的人为之沉醉；而它更是我童年记忆里最温馨的一笔，是我高中生涯里最亮丽的一程，它的草味儿，它的海风，它的排浪将永远嵌在我的脑海中。因此我要向它坦白：<br/>
棒棰岛之恋，爱你一生不变。<br/>

<strong>春游拔河</strong><br/>
一年五班            王厉        <br/>
 <br/>
阳光，美景，沙滩，春游的气氛煞是恬淡。只因有了沙滩上拔河的不平凡而昂扬似火，让我们的春游在欢笑中更加完满。<br/>
拔河的准备工作很是冗长，站在一端焦急地等待。周围积聚了众多同学，像一堵堵密不透风的墙把我们包围。人群中不时传来加油声，空气中弥漫着激动而又紧张的情绪。且看我的队友正盯着前方，双手一前一后拉住绳子，双脚在地上挖出一个小坑，双腿弯曲，俨然一个整装待发的专业队员。我也学着她的样子拉住绳子准备开拔。<br/>
大龙老师一声令下，我拉紧了绳子向后退，人群中的加油声震耳欲聋，有节奏的口号笼罩了沙滩。我的双脚在沙子中摸索，沙子却毫不配合地钻进我的鞋。我的双眼似乎什么也看不见，我的头脑也一片空白，只是在用双手死死抓住这根绳子，仿佛这就是救命稻草般。沙子飞溅，双手疲惫，似乎世界静止了，一切都凝固了。我也开始大叫，想必此刻面目狰狞，满面通红吧？我不知道什么时候才是终点，只有一个顽强的信念在脑海中涌现：“就是现在，用力！”我的双脚快速瞪着沙子，感知着胜利的到来。猛地，欢呼声爆发了。我只觉的一股强大的力量随绳而来。是的，我们赢了。我紧闭了双眼，大口地呼吸着空气中甜蜜的芬芳。我静静走出人群，丝毫没有疲惫。<br/>
坐在沙滩上，队友高兴地诉说刚刚的感受，她的目光如蓝天一样澄静。我丢起石头，，海面泛起层层涟漪，似乎对我们的喝彩。身后的人们还在为自己的班级喝彩，如朵朵团结的花儿在这里绽放。我心下澄静，双手贴近海面，幸福的语言难以言表。五班，我再一次为你自豪。你的团结，奋进如我前行的明灯，让我奋勇前进。我为你骄傲。<br/>
我抬头仰望天空，云朵飘飞，我的心底传出一声尖叫：“拔河，五班，团结，赢！”<br/>

<strong>春游随想</strong><br/>
一年五班   杨硕 <br/>
 <br/>
明明是走过许多遍的故景旧路，明明是已毫无新鲜感的海望沙滩，却为什么这一次有完全不同的感觉呢？<br/>
我家离滨海路一山之隔，假余闲日，我与父母已以双脚尺量了无数遍这条漫漫长路，也已用双眼覆盖过四季滨海路的不同风貌。当听说春游所游的是看过无数次的滨海路，我曾失望至极，叹惋久日。<br/>
而当我再游一次，正如老人手中磨转的核桃，不再有异色吸引。但我却格外兴奋，叽叽喳喳的走过了山路，眉眼带笑，蹦跳前行。<br/>
是迎春繁盛的花木更繁盛了吗？是喜鹊“戛戛”的啼声更喜庆么？还是山更青、天更澈、海更蓝，亦或枝条中仿佛爆开的生机使我震撼呢？<br/>
我觉得都不是。那么……我把目光投向身旁。只可能是人不同了吧！即使是旧景，你们也可以让它焕发出新色。<br/>
愿我们还能走过千山万水，柳岸花明。<br/>

<strong>春游小记</strong><br/>
一年五班   刘雯<br/>
原本期待着一次隆重的北大、清华之游，结果被告知那是二十四中04年之前的传统。于是我们迎来了这次滨海路踏春游。<br/>
天气真是好的没得说，久违的耀眼阳光，带来了浓郁的春意，让人全身暖洋洋的。路旁的草地早已泛起一片绿意，嫩黄的迎春，粉红的桃花绚出春天的缤纷，踏青的队伍中亦是五彩纷呈，素雅的白，深邃的黑，耀眼的粉，高贵的紫，明亮的黄和幽静的蓝交相辉映，演绎出属于五班的青春。<br/>
一路走上山顶，放眼望去，那浑然天成的碧蓝是一种无与伦比的美丽，那是属于大连的骄傲。都说下坡容易，上坡难。但在我看来，下坡更难。首先要掌握好自己的速度，不能让重力加速度任意妄为；其次要忍受下坡时给膝盖的冲击力；当然还要保持自己身体平衡。和上坡单纯的疲劳相比，下坡似乎更艰苦。<br/>
下山后便转移到棒棰岛海滩。好久没有亲迈大海了，咸咸的海风总会给生长在海边的我带来最亲切的感觉，好不惬意！中午在沙滩上举行的拔河比赛抓住了所有人的眼球。最后一场高一、高二老师间的对决，更是激起全场尖叫。第一轮，由于高一队中有两位重量级的选手，轻取高二队；第二轮，护师心切的高二学长纷纷出手相助，高二队胜！最终，比赛以1比1告终。<br/>
滨海路踏青游，抛却学业的烦恼，亲迈自然，给疲惫的身心放一天假！呵呵，真爽！<br/>

<strong>春游游记</strong><br/>
一年五班    王紫贤  <br/>
 <br/>
春风蹑着猫的脚步走来，迎春花悄悄换上嫩黄衣裳，漫山长满了绿色…..当春迈着轻盈的步伐来到人间，我们也向春天迈开脚步，开始了我们“飞扬青春”的徒步游。<br/>
几公里的环山路，和同学们一边嬉笑着，一边欣赏着沿途的风景：漫山遍野的绿是整幅画的背景，万绿丛中，一点黄的，粉的，白的小花儿，淡淡的温柔散发出无法隐藏的美丽，于是春之蓬勃生命力带来的惊喜与感动，点点滴滴流进眼里，珍存在心里。<br/>
到了棒棰岛，踩在沙滩上，呼吸着久违的海的味道，等待着即将开始的“友谊”拔河比赛。很有节奏的加油声，队员们卖力地将绳子死死拽住，我们帅气地赢下了比赛！<br/>
自由活动的时间里，同学们三五成群地围坐在一起，讲着笑话，畅谈着，天空一片湛蓝，海浪拍打沙滩的声音为我们的欢声笑语做着伴奏，春天里，沙滩上，大海边，阳光下，一切都那么温馨而美好。<br/>

<strong>棒棰岛游记</strong><br/>
<strong>1.5</strong><strong>席善博 </strong><br/>
                                                          <br/>
<strong>在鸟语阵阵中，我们的踏春旅程开始了。白云朵朵，遮不住明媚的阳光；清风习习，送来泥土的气息。郁郁葱葱的树木从两边滞后，连绵不绝的山丘敞开胸怀迎接我们的到来；旅途中的欢歌笑语使我们全然忘却了疲惫，海洋的味道引领着我们前行。</strong><br/>
<strong>最妙的是远处碧蓝的海上连接着澄澈的天空，又有一层薄薄的白雾覆盖着；白雾一会儿浓一会儿淡，有时还露出一点海水；那海却像害了羞，微微地泛起一丝波澜，荡漾着春日的美好。</strong><br/>
<strong>拖着蜿蜒的长队，我们松松散散地走着，尽情沐浴着温暖的阳光，享受着凉爽的风。平日紧绷的神经放松了，忘记了一切烦恼；此刻只有无尽的美景，和我们的欢声笑靥。越过无数了青松绿柏之后，一段下坡赫然现于眼前。此刻，最拘谨的人也忍不住开始奔跑，开始兴奋地呼喊。</strong><br/>
<strong>终于到了柔软的沙滩上，海浪拍打在沙砾上，击碎了，洒下一片白茫茫的细碎的浪花。我们纷纷打起了水漂，向大海倾诉着我们的快乐。恰恰还有拔河比赛，燃烧我们的激情。虽无运动会的紧张，但也令人心动不已；我们轻松获胜，却意犹未尽。</strong><br/>
<strong>然而美好偏偏显得短暂，于是依依不舍地，向天边的浮云挥手作别，与盘旋的海鸟道一声再见，离开了梦一样的境地。但记忆每每是永恒的，波涛的声音，永远在耳畔回荡；卵石的纹理，永远镌刻在记忆中。</strong><br/>

<strong>我与春天有个约会</strong><br/>
一年五班           孙瑞<br/>
 <br/>
今年的寒风似乎更嚣张了些，人们无奈地躲在厚厚的外衣里，看零星的几缕花儿在寒风中瑟瑟发抖。我们的心也只能悄悄地躲在学业的后面，期待着春天的到来。<br/>
这一天终于来了，同学们兴奋的笑容，操场上沸腾的气氛，还有小花园旁一抹抹的亮丽，为这一天增添了许多乐趣。我的心中早已随着花的香气飞向了目的地。因为我与春天有个约会。<br/>
温暖的阳光下，金灿灿的花儿在前方热情地向我招手。它们在这静谧的山中睡了一个冬天，刚一苏醒，便抖落到一身的萧索气息，用阳光的颜色来迎接春天，迎接它们久违的朋友。那一片黄色是那么灿烂，又那么充满柔情，让人回想起过去的那些春天。我悄悄地从它旁边走去，暗自问到：“喂，你还记得我小时候因没有力气爬山还在你面前哭闹吗？”而它们仍灿烂地笑着，好像只有这样才不枉费了这大好春光。<br/>
依旧向上爬着，依旧与这些似曾相识的朋友打着招呼，依旧细细品味着这难得的约会。一路上，同学们打打闹闹，说笑着拍影留念，想留住这难得的景色。初春的绿娇怯怯的，总隐藏在枯枝之下，等着游人细细垂询。然而，在那些艳丽的迎春花的挑逗下，它们也忍不住了，向山下望去，一层墨绿把山包裹起来，让人看上一眼便不由得安静下来。那么沉静，高雅，毫不示弱地展示着活力，尽情地拥抱春天。我呆呆地望着远处的山，暗自问到：“喂，你还记得我前年拿相机对着你拍照吗？”而山依然深沉，好像只有这样才能尽情享受春天。<br/>
走在春天里，感受春天，感悟春天。<br/>
