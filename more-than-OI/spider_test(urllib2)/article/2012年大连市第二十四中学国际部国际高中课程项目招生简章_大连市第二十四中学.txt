2012年大连市第二十四中学国际部国际高中课程项目招生简章

<td><div id="contentMidPicAD" style="float:right; clear:both; top:0; vertical-align:top;"></div>
           为在辽宁省普通高中阶段引进国际先进课程资源和科教技术，探索高中课程改革的模式，推动辽宁省基础教育的发展，建立国际人才培养基地，为辽宁省的经济发展培养具有国际竞争力的复合型人才，经辽宁省教育厅批准（辽教函[2011]202号），大连市第二十四中学与英国布罗德格林国际学校合作开办国际高中课程项目。该项目所开设的课程包括我国普通高中必修课程和剑桥国际高中课程两部分。大连市第二十四中学已于2011年4月通过了剑桥大学国际考试委员会（CIE）的审核，成为其国际考试中心，授权代码为CN572。<br/>
<br/>
    考入剑桥国际高中课程班的学生经过三年的学习，通过剑桥大学国际考试委员会（CIE）的全球统一考试后，可以根据自己的成绩和意愿在全球范围内选择申请美国、英国、澳大利亚、加拿大、新西兰、新加坡等全世界所有英语授课的大学，包括哈佛、普林斯顿、耶鲁、剑桥、牛津等世界顶级大学。<br/>
<br/>
    一、课程、学制及招生对象<br/>
<br/>
    剑桥国际高中课程（GCE A-Level）由基础课程和核心课程两部分组成，被誉为英国金牌课程体系及世界高等教育的通行证。<br/>
    基础课程：International General Certificate of Secondary Education（简称IGCSE），为核心课程的预备课程，开设有数学、英语、化学、物理、商业学等。<br/>
    核心课程：剑桥高中课程（General Certificate of Education Advanced Level，简称A-Level），源于英国全民普及的高中课程。该课程学制两年，适合顺利完成基础课程学习的学生。开设课程有数学、进阶数学、物理、化学、生物、商业学、经济学等。<br/>
<br/>
    二、文凭<br/>
<br/>
    对学习期满达到毕业要求并考核合格的学生，颁发中国辽宁省普通高中毕业证书和英国A-Level证书。<br/>
    剑桥国际高中证书即A-Level证书记录学生的考试成绩，作为学生学习能力的权威评价，已得到世界上150多个国家1100多所大学的认可。<br/>
<br/>
    三、教学管理<br/>
<br/>
    剑桥国际高中课程班由大连市第二十四中学全面管理，剑桥大学国际考试委员会（CIE）中国区高级附属中心认证的学术校长负责教学管理，剑桥外部学术委员会进行教学监督。<br/>
    剑桥国际高中课程班采用小班化授课方式。核心课程全部由具备教师资格、拥有A-Level课程教学经验的英语国家教师授课。大连第二十四中学选派优秀双语教师负责基础课程的教学。<br/>
<br/>
    四、学费<br/>
<br/>
    基础课程班每生每学年学费人民币7.5万元，核心课程班每生每学年学费人民币7.5万元。（以省物价部门审批价格为准）；以上费用不包含教材、住宿、校服、CIE全球统考考试费和国外大学升学指导服务费。<br/>
<br/>
    五、2012 年招生计划<br/>
<br/>
    2个班，60人（新高一），其中本市50人，本省外市10人；外市生源自行解决住宿问题。<br/>
             <br/>
    六、录取办法<br/>
<br/>
    1.招收本市学生采取如下方案：<br/>
<br/>
    根据大连市教育局发布的大连市2012年初中毕业升学考试和中等学校招生政策，大连二十四中学国际班为普通高中志愿的“提前录取”批次。填报志愿工作于考试前由市招生办统一组织进行。在录取阶段，中考考生英语成绩达到所需合格线后，市招生办将按照中考成绩由高分到低分确定录取名单。<br/>
<br/>
    2.招收本省外市学生采取如下方案：<br/>
<br/>
    参照2011年大连市教育局《关于大连市第二十四中学国际高中课程班有关事宜的通知》：自愿报考大连市第24中学国际高中课程班的本省外市考生，须参加当地中考考试，考试成绩原则上应达到大连市所属的省级示范性高中择校分数线标准，根据招生计划，中考考生英语成绩达到所需合格线后，将按照中考成绩由高分到低分确定录取名单。<br/>
<br/>
    3.即日起接受学生及家长咨询和本省外市考生的报名。<br/>
<br/>
