外籍学生到大连公办普通高中就学入学办理流程



<table width="100%">
<tr>
<td><div id="contentMidPicAD" style="float:right; clear:both; top:0; vertical-align:top;"></div>
<h4>
	 </h4>
<h4>
<font color="#000000">    1、通过市教育局网站或《中招考试信息》查询定点学校当年招生计划，选择有意向就读的公办普通高中</font></h4>
<h4>
<br/>
<span style="color: rgb(0, 0, 0); ">    </span><font color="#000000">2、外籍学生家长或监护人持入学所需证件材料（详见备注）于每年7月15日前到学校报名 </font></h4>
<h4>
<br/>
<span style="color: rgb(0, 0, 0); ">    </span><font color="#000000">3、学校对外籍学生证件材料进行初审，初审合格的外籍学生参加由定点学校自行组织的入学考试（相当于我市中考水平） </font></h4>
<h4>
<br/>
<span style="color: rgb(0, 0, 0); ">    </span><font color="#000000">4、学校按照外籍学生考试成绩提出拟录取外籍学生名单上报其主管教育行政部门，经主管教育行政部门核准后发放录取通知书 </font></h4>
<h4>
<br/>
<span style="color: rgb(0, 0, 0); ">    </span><font color="#000000">5、如果学生需要办理留学签证，需另外填写《外国留学人员来华签证申请表》（JW202表），由学校协助到大连市教育局、市政府外事办、市公安局出入境管理处等部门办理相关手续。 </font></h4>
<h4>
<br/>
<font color="#000000">备注：外籍学生到大连中小学就学，需要提供相应证件材料，分两种情况：</font></h4>
<h4>
<br/>
<font color="#000000">　　1、父母在我市投资或正式就职并在我市居住的外籍学生，需要提供：<br/>
	　　①父母护照及职业签证 ②父母及学生在连居住证明原件及复印件一份（A4纸复印）③父母在连投资证明或就职证明 ④学生护照原件及复印件一份（A4纸复印） ⑤中国公证机关签发的亲子关系公证文书 ⑥《大连市中小学外籍学生入学申请表》一式四份（可从所在区教育局获取或通过“大连市教育局网站-网上办公-表格下载”下载，用A4纸打印）。</font></h4>
<h4>
<br/>
<font color="#000000">　　2、以留学为目的到我市中小学就学的外籍学生，如父母在连常住，需要提供：<br/>
	　　①父母护照 ②学生护照原件及复印件一份（A4纸复印）③父母及学生在连居住证明原件及复印件一份（A4纸复印）④中国公证机关签发的亲子关系公证文书 ⑤《大连市中小学外籍学生入学申请表》一式四份；<br/>
	　　如父母不在连常住委托在连常住人员作为监护人，需要提供：<br/>
	　　①学生护照 ②学生和委托监护人在连居住证明原件及复印件一份（A4纸复印）③中国公证机关签发的委托监护公证文书 ④委托监护人的身份证明原件及复印件一份（A4纸复印） ⑤《大连市中小学外籍学生入学申请表》一式四份</font></h4>
