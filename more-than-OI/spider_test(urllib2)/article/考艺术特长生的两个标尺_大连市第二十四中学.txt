考艺术特长生的两个标尺

<td><div id="contentMidPicAD" style="float:right; clear:both; top:0; vertical-align:top;"></div>
      　　随着各地艺术特长生报考政策出台，特长生报名也提到日程上来。孩子是否适合报考艺术特长生?是很多考生家长此刻最急切想要知道的。 <br/>
<br/>
　　目前各招生院校招收艺术特长生的项目主要包括：音乐、舞蹈、戏剧、书画。因此，想要报考艺术特长生，孩子最起码应该具备其中一方面的艺术特长。 <br/>
<br/>
　　是不是孩子具备一定特长，就一定适合报考艺术特长生呢?并非如此。考生家长还应参考省市及各校发布的招生办法和招生简章，以此作为标尺进行衡量。 <br/>
<br/>
　<strong>　标尺一：看清当地报考政策 </strong><br/>
<br/>
　　考生家长首先要清楚本地区的艺术特长生报考政策。 <br/>
<br/>
　　近几年，有的地区直接在报考要求里，限定报考艺术特长生的考生必须是当地学生艺术团成员，或者参加全国、市级比赛成绩优秀者;而有的地区则只要考生符合当年高考报考要求，就可以报考艺术特长生。当然，即便孩子符合当地的报考条件，也需要参加统测或校测才能取得资格。所以，孩子是否够格参加艺术特长生考试，把握好政策至关重要。 <br/>
<br/>
　<strong>　标尺二：分析院校招生简章 </strong><br/>
<br/>
　　院校招收艺术特长的目的是为了活跃校园文化生活，所以除当地当年的招生办法会对艺术特长生报考提出要求，招生院校要求也各不相同。 <br/>
<br/>
　　像清华大学2012年艺术特长生报名条件里，就对艺术特长生应该具备哪些条件做了规定：“音乐、表演类所有考生具备下述条件之一：取得各类社会考级最高级别证书;中学阶段有两年以上校级(含)以上学生艺术团活动经历;中学阶段参加过地市级(含)以上比赛并获奖者;中学阶段参加过地市级(含)以上大型演出活动。” <br/>
<br/>
　　北京理工大学虽然也提出要有相关考级证书，但同时表示：无相关考级证书但具有同等水平的考生亦可报名。 <br/>
<br/>
　　北京师范大学2012年艺术特长生招生简章里，则没有限定考生必须出具相关艺术特长证明。 <br/>
<br/>
　　除此之外，考生家长一定要注意查看孩子所报考项目的招生要求。具体到每个招生项目时，不同院校的报考条件也有差别。 <br/>
<br/>
　　例如，清华大学的舞蹈类项目要求：男生身高173cm以上，女生身高163cm以上。而北京理工大学则要求：报考舞蹈项目的考生，女生身高需1.60米以上，男生身高需1.70米以上，参加过大型演出及比赛获奖者可注明。 <br/>
<br/>
　　总之，考生家长一定要细读各院校的招生简章，尤其要分析孩子想要报考的艺术项目要求，选择符合孩子自身条件的艺术特长项目和院校。 <br/>
<br/>
　　同时，考生家长还要注意一点：按教育部规定，目前艺术特长生招生人数控制在招生院校本年度招生计划总数的1%以内。2012年北京大学全国总招生人数3495人，而艺术特长生拟定招收人数30名，仅占总招生人数的0.86%。如果孩子的艺术特长符合目标院校的各项招生要求，孩子单凭文化课成绩考取目标院校有点儿难度时，那么，他适合报名参加该所院校的艺术特长生考试。倘若艺术特长生加分，对孩子能否考上目标院校影响不大，就不要浪费时间和精力了。<br/>
